# About Page
**Path**: `src/app/about/page.tsx`

## Purpose
Static informational page displaying details about the shop.

## Features
*   **Tenant Info**: Displays shop name and phone number fetched from `api.shop.getShopDetails`.
*   **Simple Layout**: Dark themed UI consistent with the brand.

## Dependencies
*   `~/trpc/server`: `api`
*   `next/link`: `Link`
