# Shifts Page
**Path**: `src/app/pos/shifts/page.tsx`

## Purpose
Management of cashier shifts and cash reconciliation.

## Features
*   **Shift Control**: Open new shift (enter starting cash), Close current shift (enter closing cash).
*   **History**: View past shifts.
*   **Navigation**: Link back to POS terminal.

## Dependencies
*   `~/trpc/react`: `api` (pos router for shift management).
