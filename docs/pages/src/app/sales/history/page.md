# Sales History Page
**Path**: `src/app/sales/history/page.tsx`

## Purpose
Reporting view for completed sales history.

## Features
*   **List**: Table of historical sales.
*   **Aggregates**: Displays Total Revenue.
*   **Tools**: Search functionality, Export to PDF/Excel.

## Dependencies
*   `~/trpc/react`: `api` (sales router).
*   `~/lib/export-utils`: Export logic.
